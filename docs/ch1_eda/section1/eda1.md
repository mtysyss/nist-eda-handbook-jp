# 1.1. EDA入門  
 
#### 概要  

探索的データ解析とは何か。その起源はいつか。どのように、どこで生まれたのか。古典的とベイズなど他のデータ解析手法とどう異なるのか。EDAは統計的図表現と同義か。統計的図表現はEDAにおいてどのような役割を果たすのか。統計的図表現はEDAと同一か。  

これらの疑問と関連する論点を本節で扱う。本節ではこれらの疑問に答え、 EDAの仮定、原則、技法に関する必要な参照される枠組みを説明する。
