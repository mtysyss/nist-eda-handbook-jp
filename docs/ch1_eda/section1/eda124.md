# 1.1.2.4. 厳格さ

#### 古典的
古典的技法は科学と工学の確率論的基盤として機能する。古典的技法の最も重要な特徴は、厳密で形式的且つ「客観的」なことである。

#### 探索的
EDA技法はその厳密さや形式性を共有しない。EDA技法は、適切なモデルがどのようなものであるかについて非常に示唆的であり、指示的であり、洞察に満ちていることで、その厳密さの欠如を補っている。

EDA手法は主観的であり、解釈に依存するが、この解釈は分析者によって異なることもあるが、経験豊富な分析者は通常、同じ結論に到達する。
