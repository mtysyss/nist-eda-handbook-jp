# 1.1.3. 探索的データ解析と要約分析との相違

#### 要約

要約分析とは、単に過去のデータ集合を数値的に集約したものである。
これは極めて受動的であり、焦点は過去に置かれる。
その目的は往々にして、いくつかの主要な統計量（例えば平均値や標準偏差）を導き出すことにあり、それらはデータ集合に代わる形もしくは要約表の形でデータ集合に追加される。

#### 探索的

対照的に、EDAの最も広範な目標は、データの裏側にある工学／科学過程への洞察を得ることにある。
要約統計量が受動的で履歴的であるのに対し、EDAは能動的で未来志向である。
過程を「理解」し将来的に改善しようとする試みにおいて、EDAはデータを「窓」として用い、そのデータを生み出した過程の核心を覗き見るのである。
研究・製造の世界において要約統計量には記録保存の役割があるが、EDA手法にはそれよりもはるかに大きな役割がある。
