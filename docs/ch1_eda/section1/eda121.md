# 1.1.2.1. モデル

#### 古典的

古典的な手法では、データに対してモデル（決定論的および確率論的の両方）を適用する。決定論的モデルには、例えば
[回帰モデル](../ch4_pmd/section1/pmd141.md)や[分散分析（ANOVA）](../section3/eda355.md)モデルが含まれる。最も一般的な確率論的モデルは、決定論的モデルに関する誤差が正規分布に従うと仮定する——この仮定はANOVAのF検定の妥当性に影響を与える。

#### 探索的

探索的データ解析（EDA）手法は、データに対して決定論的や確率論的モデルを強制しない。むしろ、EDA手法は、データに最も適合する許容可能なモデルをデータが自ら示唆することを可能にする。
