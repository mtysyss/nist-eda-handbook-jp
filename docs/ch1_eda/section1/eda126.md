# 1.1.2.6. 前提条件

#### 古典的

古典的手法の「良い知らせ」は、古典的技法に基づく検定が通常非常に感度が高いことである。つまり、例えば真の位置の偏位が生じたとき、こうした検定はしばしばその偏位を検出する検出力を持っており、その偏位が「統計的に有意」であると結論づけることができる。 「悪い知らせ」は、古典的検定が基礎となる仮定（正規性など）に依存するため、検定の結論の妥当性が基礎仮定の妥当性に依存してしまう点である。 さらに悪いことに、分析者がその仮定を正確に把握していない場合や、把握していても検証していない場合がある。したがって科学的結論の妥当性は、前提条件の妥当性と本質的に結びつく。実際、こうした前提条件が不明もしくは未検証の場合、科学的結論の妥当性は疑わしい。

#### 探索的

多くのEDA技法の前提条件は少ないか全く仮定を置かない。それらはデータを先入観や制約に縛られることなく、「あるがまま」の姿ですべて提示し、可視化する。
