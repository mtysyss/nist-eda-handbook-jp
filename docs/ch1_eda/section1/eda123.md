# 1.1.2.3. 技法

#### 古典的

古典的技法は一般的にその特性から[定量的](../section3/eda35.md)である。
これには、[ANOVA](../../ch7_prc/section4/prc42.md)、[t検定](../section3/eda353.md)、[カイ二乗検定](../section3/eda358.md)、[F検定](../section3/eda359.md)が含まれる。

#### 探索的

EDA技法は一般的に[図表現](../section3/eda33.md)である。これらには[散布図、特性プロット](../section3/scatterp.md)、[箱ひげ図](../section3/boxplot.md)、[ヒストグラム](../section3/histogra.md)、[2元ヒストグラム](../section3/bihistog.md)、[確率プロット](../section3/probplot.md)、[残差プロット](../section3/6plot.md)、[平均プロット](../section3/meanplot.md)が含まれる。
