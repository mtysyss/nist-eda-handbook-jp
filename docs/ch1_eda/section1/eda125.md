# 1.1.2.5. データ処理

#### 古典的

古典的な推定び技法は、全データを処理し、それを少数の要素（「推定値」）に変換する特徴を持つ。これは長所でもあり短所でもある。長所は、これらの少数の要素が母集団の重要な特性（位置、ばらつきなど）に焦点を当てる点である。 欠点は、これらの少数の特性に集中することで、同じ母集団の他の特性（歪度、裾の長さ、自己相関など）が除外されてしまう可能性がある点である。この意味で、この「選別」過程によって情報が失われる。

#### 探索的

一方、EDA手法では、利用可能なデータ全体を活用し（且つ示す）ことが多いため、この意味において情報の損失は生じない。
