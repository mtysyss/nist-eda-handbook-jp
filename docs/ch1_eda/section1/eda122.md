# 1.1.2.2. 焦点

#### 古典的
両者の手法の焦点は大きく異なる。古典的解析では、モデルそのものに焦点が当てられる——モデルの母数を推定し、モデルが予測値を与える。

#### 探索的
探索的データ解析では、データの構造、外れ値、データが示唆するモデルといったデータそのものに焦点が当てられる。
