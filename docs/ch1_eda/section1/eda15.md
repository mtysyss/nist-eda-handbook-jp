# 1.1.5. 図表現の役割

#### 定量的と図表

統計およびデータ解析の手順は、大きく分けて二つの部分に分けることができる：

* [定量的](../section3/eda35.md)
* [図表的](../section3/eda33.md)

#### 定量的
定量的技法とは数値や表形式の出力を得る一連の統計的手続きである。定量的技法の例には以下が含まれる。

* [仮説検定](../section3/eda35.md#TESTS)
* [分散分析](../../ch7_prc/section4/prc42.md)
* [点推定と信頼区間](../section3/eda35.md#INTERVAL)
* [最小二乗回帰](../../ch4_pmd/pmd.md)

これらと類似の技法はすべて価値があり、古典的解析の観点では主流である。

#### 図表
一方、統計ツールには一般的に図表現技法と呼ばれる膨大な蓄積がある。これには以下が含まれる：

* [散布図](../section3/scatterp.md)
* [ヒストグラム](../section3/histogra.md)
* [確率プロット](../section3/probplot.md)
* [残差プロット](../section3/6plot.md)
* [箱ひげ図](../section3/boxplot.md)
* [ブロックプロット](../section3/blockplo.md)

#### EDA手法の図表現技法への強い依存
EDA手法はこれらおよび類似した図表現技法に強く依存している。
図表現の手順は、単にEDAの文脈で利用できるツールではなく、必ず使用すべきツールである。
このような図表現のツールは、次の観点からデータ集合への洞察を得るための最短経路である。

* 仮説検定
* モデル選択
* モデル検定
* 推定量選択
* 関係性の特定
* 因子効果判定
* 外れ値検出

統計的図表現を使用しない場合、データの基礎になっている構造の一つ以上の側面に対する洞察を放棄していることになる。
