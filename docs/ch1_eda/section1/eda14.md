# 1.1.4. EDAの目標

#### 主目標と二次目標

探索的データ解析（EDA）の主な目標は、データ集合とその基盤構造に対する分析者の洞察を最大化すると同時に、分析者がデータ集合から抽出したいと考える具体的な項目をすべて提供することである。
例えば：

1. 適合度の高い簡潔なモデル
2. 外れ値の一覧
3. 結論の頑健性の把握
4. 母数の推定値
5. それら推定値の不確かさ
6. 重要因子の順位付け一覧
7. 個々の要因が統計的に有意であるか否かに関する結論
8. 最適な設定値

#### データへの洞察

洞察とはデータの基礎構造を検知し解明することを意味する。
そうした基礎構造は、上記の項目一覧に網羅されているとは限らない。
各項目は解析の具体的対象となるが、真の洞察とデータ集合への「感覚」は、分析者がデータの様々な微妙な側面を慎重に探求し掘り下げる過程で得られる。 
データの本質を捉える「感覚」は、完全に様々なグラフィカル手法の適用から得られる。
これらの手法の集合こそが、データの本質を覗き見る窓となる。
図表現は代替不可能である。つまり、適切に選択された図表現と同等の洞察を与える定量的な代替手段は存在しない。 

データに対する「感覚」を得るには、分析者がデータに何が含まれているかを知るだけでは不十分である。
分析者はデータに何が含まれていないかも知る必要があり、それを実現する唯一の方法は、データに適用される一連の慎重な図表現技法の文脈において、私たち人間自身が持つパターン認識能力と比較能力を活用することである。
