# 1.2.3. 仮定を検定する技法

#### 科学および工学の結論の妥当性を保証するのに役立つ基礎となる仮定の検定  
最終的な科学的／技術的結論の妥当性は、基礎となる単変量の仮定の妥当性と不可分に関連しているため、上記4つの仮定のそれぞれを日常的に検定することが真に必要なことは当然の帰結である。

#### 前提となる仮定を検定する4つの技法
以下の探索的データ解析（EDA）技法は、基礎となる仮定の日常的な検定に簡便で効率的かつ強力なものである。

1. [連シーケンス・プロット](../section3/runseqpl.md)（ $$Y_i$$ 対 $$i$$ ）
2. [ラグ・プロット](../section3/lagplot.md)（ $$Y_i$$ 対 $$Y_{i-1}$$ ）  
3. [ヒストグラム](../section3/histogra.htm)（ 計数 対 $$Y$$ の区間群 ）  
4. [正規確率プロット](../section3/normprpl.md)（ 順序付き $$Y$$ 対 理論的順序付き $$Y$$ ）  

#### データを素早く特徴付けるための単ページのプロット
4つのEDAプロットを並べて表示することで、データの特性を素早く確認できる。以下のプロットは次の順序で並べられている。

1. 連シーケンス・プロット - 左上  
2. ラグ・プロット - 右上  
3. ヒストグラム - 左下  
4. 正規確率プロット - 右下  

#### プロット例：仮定が成立
![](https://www.itl.nist.gov/div898/handbook/eda/gif/randn4p.gif) 

[500個の正規乱数列](https://www.itl.nist.gov/div898/handbook/datasets/RANDN.DAT)のこの[4-プロット](../section3/4plot.md)は、位置が固定され、ばらつきが固定され、無作為であり、明らかに固定された正規分布に近似され、外れ値が存在しない過程を示している。 

#### プロット例：仮定が成立しない
4つの基礎的な仮定のうち1つ以上が成立しないとき、[以下の例](https://www.itl.nist.gov/div898/handbook/datasets/LEW.DAT)に示す様々なプロットにその状態が現れる。

![](https://www.itl.nist.gov/div898/handbook/eda/gif/lew4p.gif)  

この[4-プロット](../section3/4plot.md)は、固定された位置、固定されたばらつき、非無作為性（振動的）、正規分布ではないU字型分布、そして複数の外れ値を持つ過程を明らかにしている。
