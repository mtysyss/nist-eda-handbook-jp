# 1.2.1. 基礎となる仮定

#### 測定過程に内在する仮定
あらゆる測定過程には通常、4つの仮定が存在する。
すなわち、当該過程から得られるデータが「以下のように振る舞う」という仮定である。

1. 無作為抽出であり
2. 固定された分布から
3. 分布は固定された位置を持つ 
4. 分布の分散が固定である

#### 単変量もしくは単一応答変数
上記3項で言及された「固定位置」は、問題の種類によって異なる。
最も単純な問題の種類は単変量、すなわち単一の変数である。
単変量問題における一般的なモデル

$$応答 = 決定論的成分 + 無作為成分$$

は次式になる。

$$応答 = 定数 + 誤差$$

#### 単変量モデルの仮定
この場合、「固定された位置」とは単純に未知の定数である。
したがって、我々は目の前の過程が一定条件下で動作していると想定できる。
その条件下では、単一のデータ列が作られ、以下の性質を持つ。

* データ同士が無相関である  
* 無作為成分が固定分布を持つ  
* 決定論的成分は定数のみで構成される
* 無作為成分のばらつきは固定されている

#### 多変数関数への外挿
単変量モデルの普遍的な能力と重要性は、その決定論的成分が単なる定数ではなく、実際には多くの変数の関数である、より一般的な場合へ容易に拡張可能であり、工学的な目的はその[関数の特徴づけモデル化](../../ch4_pmd/pmd.md)にある。

#### 残差は単変量仮定に従って振る舞う
重要な点は、要因の数や関数の複雑さにかかわらず、技術者が適切なモデルを選択できれば、生データと適合モデルによる予測値の差（残差）自体が単変量過程として振る舞うべきということである。
さらに、この単変量過程による残差は次のように振る舞う。

* 無作為抽出であり
* 固定された分布から
* 固定された位置（この場合は0）を持ち
* 固定された分散を持つ

#### モデルの検定
したがって、[適合モデルの残差](../../ch4_pmd/section4/pmd44.md)が、実際に理想的な振る舞いを示す場合、基礎となる仮定の検定は、選択されたモデルの妥当性と適合度の評価手段である。
一方、選択された適合モデルからの残差が上記単変量の仮定の一つ以上を満たさない場合、そのモデルは不適切であり、モデルを改良する余地がある。
