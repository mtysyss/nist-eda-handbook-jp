# 1.2. EDAの前提条件

#### 概要
科学技術実験の領域は事実上無限である。この多様性の海において、分析者が体系的かつ有効に、支持可能で再現性のある研究の結論に到達することを可能にする共通基盤は存在するのか。

幸いにも、そのような基盤は存在し、それはあらゆる測定過程が複雑であろうとも、根底にある仮定を有するという事実に根ざしている。
本節では、それら仮定の内容、重要性、検定方法、そして仮定が成り立たない場合の結果について扱う。
