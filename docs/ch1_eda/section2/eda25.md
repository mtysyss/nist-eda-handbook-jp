# 1.2.5. 導かれる結果

#### 仮定が成り立たない場合
基礎となる仮定の一部が成り立たない場合、どう対処すべきか？
どのような是正措置を講じられるか？
この問題に前向きに取組む方法は、基本的仮定の検定を過程の理解のための枠組みと捉えることである。
仮定の検定は、そうでなければ表面化しなかったかもしれない過程に関する重要な側面への洞察を促す。

#### 第一の目標：正確かつ妥当な科学的結論
主な目的は、分析から導き出される科学的 / 技術的結論が正確で検定済みかつ完全であることである。
これには通常、適合性の高いモデルの導出や現実的な母数推定値の計算といった中間目標が含まれる。
常に最終目標として、「過程を動かす要因」に対する理解と「感覚」を含むべきである。
経験豊富な科学者・技術者と、興味深い「異常」や特性に満ちたデータ集合を結びつけることほど、発見を促す強力な触媒は存在しない。
