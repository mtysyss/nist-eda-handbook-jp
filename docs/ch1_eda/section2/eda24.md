# 1.2.4. 4-プロットの解釈

#### EDAプロットの解釈：平坦で等間隔、無作為、ベル型、直線型
前ページで論じた[4つのEDAプロット](../section2/eda23.md)は、以下の基本的仮定を検定するために用いられる。

1. **固定位置：**  
   固定位置の仮定が成り立つ場合、実行順序プロットは平坦で編位していない。
2. **固定ばらつき：**  
     固定ばらつきの仮定が成立する場合、連シーケンス・プロットの垂直方向の広がりは、水平軸全体でほぼ均一になる。
3. **無作為性：**  
       無作為性の仮定が成り立つ場合、ラグ・プロットは構造を持たず無作為になる。
4. **固定分布：**  
         固定分布の仮定が成立する場合、特に固定正規分布が成立する場合、
    1. ヒストグラムはベル型になり
    2. 正規確率プロットは直線的になる。

#### 仮説を検定するために利用するプロット
逆に、[基礎となる仮定](https://www.itl.nist.gov/div898/handbook/eda/section2/eda21.md)はEDAプロットを用いて検定する。

* **連シーケンス・プロット：**  
  連シーケンス・プロットが平坦で編位がない場合、固定位置の仮定が成立する。連シーケンス・プロット全体でほぼ均一な垂直方向の広がりがある場合、固定変動の仮定が成立する。
* **ラグプロット：**  
    ラグプロットに構造が認められない場合、無作為性の仮定が成立する。 
* **ヒストグラム：**  
      ヒストグラムがベル型であれば、基礎分布は対称であり、近似的に正規分布である。 
* **正規確率プロット：**  
        正規確率プロットが直線的であれば、基礎分布はおおむね正規分布である。

これら4つの仮定がすべて成立する場合、その過程は定義上「統計的管理下」にあるとされる。
