# 1.2.2. 重要性

#### 予測可能性と統計的管理
予測可能性は科学と工学において極めて重要な目標である。
4つの基本的仮定が成り立つならば、確率的予測可能性を満たしたことになる。
つまり、過去の過程だけでなく、将来の過程についても確率的記述である能力である。
要するに、そのような過程は「統計的管理下にある」と言われる。

#### 技術的結論の妥当性
さらに、4つの仮定が有効であれば、その過程は、有効な科学的・工学的結論を導くのに適している。
4つの仮定が無効であれば、その過程は（位置、変動、または分布に関して）編位している状態にあり、予測不可能で、制御不能である。
このような過程を位置の推定値、ばらつきの推定値、あるいは分布の「推定値」によって単純に特徴づけることは、当然のこととして有効ではなく、（科学的または法的に）支持されず、実験室で再現不可能な工学的な結論へと導くことになる。
