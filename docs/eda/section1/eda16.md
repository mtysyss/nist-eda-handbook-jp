# 1.1.6. EDA/図表現の例

#### アンスコムの例
データ集合への洞察を提供する上で、図表が果たす中心的な役割を示す、シンプルで古典的な（[アンスコムの](../ch1_eda/section4/eda43.md)
）例は、

#### 要約統計量

分析の目的が要約統計量の計算に加え、X を関数とする Y の最良線形近似を決定するとき、結果は以下のように示される。

$N = 11$

$Xの平均値 = 9.0$
$Yの平均値 = 7.5$
$切片 = 3$
$傾き = 0.5$
$残差標準偏差 = 1.237$
$相関係数 = 0.816$

上記の定量分析は有用ではあるが、データに関する洞察は限定的である。 

#### 散布図

対照的に、データの単純な以下の[散布図](../ch1_eda/section3/scatterp.md)
![アンスコムデータの散布図](https://www.itl.nist.gov/div898/handbook/eda/section1/gif/anscomb1.gif)

は次のことを示唆している。

1. データ集合は散らばりはあるものの線形曲線の「ように振る舞う」
2. より複雑なモデル（二次関数など）を採用する根拠はない
3. 外れ値は存在しない
4. データの垂直方向の広がりは、X値に関わらず高さが等しく見える。これはデータ全体が等しい精度であることを示しており、したがって「通常の」（すなわち、均等加重）の近似が適切である。 

#### ３つの追加データ集合

この種のデータの特徴付けは、データに対する洞察や感覚を得るための核心である。
そのような洞察や感覚は定量的統計量から得られるものではない。
むしろ、切片や傾きといった定量的統計量の計算は特徴付けの後に行われるべきであり、特徴付けが真実である場合にのみ意味を持つ。 図表による洞察の段階を省略したときに生じる情報の損失を説明するため、以下の3つのデータ集合[[アンスコムのデータ集合2、3、4](../../datasets/ANSCOMBE.DAT)を検討する。

#### データ集合2の定量的統計量

データ集合2の定量分析の結果は以下の通りである。

$N = 11$
$Xの平均 = 9.0$
$Yの平均 = 7.5$
$切片 = 3$
$傾き = 0.5$
$残差標準偏差 = 1.237$
$相関係数 = 0.816$

これはデータ集合1の分析結果と同一である。統計量が示す通り、2つのデータ集合は「同等」であると単純に考えがちであるが、統計量が教えてくれないことは何か。

#### データ集合3および4の定量的統計量

驚くべきことに、データ集合3と4の定量分析でも以下の結果が得られた。

$N = 11$
$Xの平均 = 9.0$
$Yの平均 = 7.5$
$切片 = 3$
$傾き = 0.5$ 
$残差標準偏差 = 1.236$
$相関係数 = 0.816 (データ集合4では0.817)$

これは、ある種の定量的観点において、4つのデータ集合全てが「同等」であることを示唆している。
しかし実際には、4つのデータ集合は「同等」とは程遠く、どのデータ集合のあらゆる探索的データ解析手法の第一歩である散布図を見ても、そのことは即座に明らかにである。

#### 散布図の解釈

散布図からの結論は以下の通り。

1. データ集合1は明らかな直線性を示し、散らばりがある
2. データ集合2は明らかに二次関数的な関係を示している
3. データ集合3には明らかに外れ値が存在する
4. データ集合4は明らかに実験設計の不備による被害を受けており、データの大部分から大きく離れた単一データ点が「しっぽが犬を振る（訳註：些細な理由から大きな決定を左右する）」状態になっている。

#### 探索的解析の重要性

これらの点はまさに、データ集合に対する「洞察」と「感覚」を与える本質であり、定義づけるものである。
これらはデータに対する開かれた探索的データ解析（EDA）手法の目標であり、その成果でもある。
定量的統計量自体は間違ってはいないが、不完全である。
不完全である理由は、それらが数値的**要約**であり、要約操作においてデータを数個の数値に適切に削減することで、データの特定の側面（位置、切片、傾き、関連性の度合いなど）に焦点を当てる点で優れた働きをするからである。
この過程ではデータが**抽出され**、焦点を当てる操作において必然的に他の重要な情報が省略、選別される。
定量的統計量は焦点を当てるが同時に抽出も行う。この抽出こそが、定量的手法を最良でも不完全、最悪では誤解を招くものにしている。
データ集合2、3、4の推定された切片（=3）と傾き（=0.5）は誤解を招く。なぜなら推定は仮定された線形モデルの文脈で行われ、この線形性の仮定こそがこの分析の致命的な欠陥だからである。
